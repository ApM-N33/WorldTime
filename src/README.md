# WorldTimesTS 🌍

WorldTimesTS is a TypeScript-based web application for managing and visualizing time across different time zones.

The project allows users to interact with time-related tools such as world time tracking, stopwatch, time conversion, and an interactive time zone map.

Built with modern web technologies and optimized for smooth UI interactions and animations.

---

## 🚀 Features

### 🌎 World Time
- Add cities to track their local time
- Display date and time for selected cities
- Manage and remove city cards
- Smooth UI animations

---

### ⏱ Stopwatch
- Stopwatch functionality similar to iOS
- Start, stop, and reset timer
- Lap (circle) tracking
- Real-time updates

---

### 🔄 Time Converter
- Compare time between two cities
- Convert time values across time zones
- Synchronize different local times

---

### 🗺 Time Zone Map
- Interactive world map
- Zoom in / zoom out functionality
- Click on countries to add them
- Display country time zones
- Show local time & date per country
- Manage multiple selected countries

---

## Tech Stack

- TypeScript
- Vanilla Js
